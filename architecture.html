Key Data Flows
LOGIN
login.html → POST /api/auth/login → JWT → localStorage → all protected routes
UPLOAD
dashboard.html → POST /api/documents/upload → disk + DB row + risk scoring
SHARE
Creates Share record with share_token · expiry · custom_masks JSON
VIEW
shared-view.html → GET /api/shares/access/{token}/watermarked → watermarked bytes served
ALERT
Access anomaly → Alert row → Admin panel notified → Bland.ai voice call auto-triggered
LOCK
User says NO on voice call → backend detects transcript → account locked automatically
